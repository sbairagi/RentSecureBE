## Usage

These documents are referenced by:

- `README.md` — project-level AI governance pointer
- `docs/README.md` — documentation index
- `.kilo/instructions/*` — AI session instructions
- `docs/ai-governance/AI-Architecture-Review.md` — cross-references ADRs

AI governance documents are maintained by the AI Governance Lead. Changes require approval from the Platform Architect and Tech Lead.
