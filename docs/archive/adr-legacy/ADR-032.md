# Free Notification Channels Only in Year 1

## Status
Accepted

## Date
2026-07-14

## Context
RentSecure needs to send notifications to users during Year 1. Paid notification channels (WhatsApp Business API, SMS providers) introduce per-message costs that would exceed the ₹2,000–3,000 INR monthly budget, especially as the user base grows. Additionally, managing WhatsApp Business API approval and SMS provider integrations adds operational complexity during the bootstrap phase.

## Decision
Year 1 uses only free notification channels:
- Email (via Django SMTP or AWS SES Free Tier: 62,000 emails/month free)
- Firebase Cloud Messaging (FCM) for push notifications (free)
- In-App Notifications (database-backed, free)

WhatsApp and SMS adapters are implemented with full provider abstraction but remain disabled behind feature flags. UI displays "Coming Soon" badges for these channels.

## Consequences
- Zero notification costs within Year 1
- No dependency on paid notification providers
- Reduced operational complexity
- Limited real-time delivery compared to WhatsApp/SMS
- Triggers Stage 2 migration based on user feedback or email engagement metrics
