# Kilo Instruction Library

Modular instruction documents for RentSecure Kilo sessions.

## Loading Order

Instructions are loaded in the order listed in `kilo.jsonc` `instructions` array.

1. `README.md` — meta-context and index
2. `universal.md` — universal project rules reference
3. `architecture.md` — architecture constraints
4. `backend.md` — backend specifics
5. `security.md` — security guardrails
6. `testing.md` — testing requirements
7. `frontend.md` — frontend specifics
8. `notifications.md` — notification module
9. `finance.md` — finance/payment module
10. `smartbot.md` — AI assistant module
11. `onboarding.md` — onboarding guidance
