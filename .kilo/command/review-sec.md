# review-sec

Execute security-focused code review.

## Configuration

```jsonc
{
  "command": {
    "review-sec": {
      "template": "Run security review on changed files",
      "description": "Execute security-focused code review",
      "agent": "plan",
      "subtask": true
    }
  }
}
```

## Usage

Invoke the `review-sec` command in a Kilo session.
