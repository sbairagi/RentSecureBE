# Architecture Reports

This directory is the intended destination for architecture-generated reports.

## Current State

The following architecture report files currently exist at the repository root and are **not** moved during Phase 0 because they are generated by CI scripts:

- `architecture-compliance-report.json`
- `architecture-compliance-report.md`
- `architecture-dependency-graph.dot`
- `architecture-dependency-graph.mmd`
- `architecture-summary.txt`

These files are generated by:
- `scripts/architecture_contract.py`
- `.github/workflows/architecture-guard.yml`
- `.github/workflows/architecture.yml`

Moving them would break CI artifact paths and script references.

## Intended Migration

During a future architecture phase, the CI scripts and GitHub Actions will be updated to write reports to this directory (`architecture/reports/`), at which point these files will be moved here without content modification.

Until then, treat this directory as the canonical location for architecture reports and update generation scripts accordingly when the time comes.

## Report Index

| Report | Description | Generated By |
|--------|-------------|--------------|
| `architecture-compliance-report.json` | Machine-readable compliance score | `architecture_contract.py` |
| `architecture-compliance-report.md` | Human-readable compliance report | `architecture_contract.py` |
| `architecture-dependency-graph.dot` | Graphviz dependency graph | CI workflow |
| `architecture-dependency-graph.mmd` | Mermaid dependency graph | CI workflow |
| `architecture-summary.txt` | Compliance summary text | CI workflow |
